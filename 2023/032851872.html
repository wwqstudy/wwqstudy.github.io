简单回文数判断：

初级：

num=input("请输入一个正整数：")
length=len(num)
number=int(num)
flag=True
if number<0:
    print("不是回文数。")
else:
    for i in range(length//2+1):
        if num[i]==num[length-i-1]:
            continue
        else:
            flag=False
    if flag:
        print("是回文数。")
    else:
        print("不是回文数。")

进阶：

num=input("请输入一个正整数：")
if num==num[::-1]:
    print("是回文数")
else:
    print("不是回文数")
或者
num=input("请输入一个正整数：")
print("是回文数") if num==num[::-1] else print("不是回文数")

字符串Ⅴ，Ⅵ

格式化字符串：

占位
"现在是{}年".format(year)
"现在是{}年{}月{}日".format(year,month,day)

位置索引
"现在是{0}年{2}月{1}日".format(year,day,month)

关键字
"现在是{year}年{month}月{day}日".format(year=2023,month=4,day=18)

注释
"现在是{year}年{month}月{{}}日".format(year=2023,month=4)

格式方法：

^, <**, **> 分别是居中、左对齐、右对齐，后面带宽度， : 号后面带填充的字符，只能是一个字符，不指定则默认是用空格填充。

+ 表示在正数前显示 **+**，负数前显示 **-**； （空格）表示在正数前加空格

b、d、o、x 分别是二进制、十进制、八进制、十六进制。

F(f)-string方法：

year,month,day=2023,4,18
F"现在是{year}年{month}月{day}日"
结果：'现在是2023年4月18日'

小菜：

enumerate()函数：

含义：返回一个枚举对象，功能是将可迭代对象中的每个元素及从0开始的序号共同构成一个二元组的列表(以迭代器的形式存在)

eg：
x="FishC"
enumerate(x)
输出：<enumerate object at 0x000001E0CA980E00>
list(enumerate(x))
输出：[(0, 'F'), (1, 'i'), (2, 's'), (3, 'h'), (4, 'C')]

zip()函数：

含义：用于创建一个聚合多个可迭代对象的迭代器，它会将作为参数传入的每一个可迭代对象的每一个元素依次组合成元组

eg:
x=[1,2,3]
y=[1,2,3]
zippad=zip(x,y)
zippad
输出：<zip object at 0x000001E0CA9CB1C0>
list(zippad)
输出：[(1, 1), (2, 2), (3, 3)]

**参数只要是序列就行**

eg:
x=[1,2,3]
y="FishC"
zippad=zip(x,y)
zippad
输出：<zip object at 0x000001E0CA9CB1C0>
list(zippad)
输出：[(1, 'F'), (2, 'i'), (3, 's')]#此时y数据并未完全输出

#	y数据完全输出
x=[1,2,3]
y=[1,2,3,4]
import itertools
zippad=itertools.zip_longest(x,y)
list(zippad)
输出：[(1, 1), (2, 2), (3, 3), (None, 4)]

map()函数：

含义：map()函数会根据提供的函数对指定的可迭代对象的每个元素进行运算，并将返回运算结果的迭代器

eg:
mappad=map(ord,"FishC")
list(mappad)
输出：[70, 105, 115, 104, 67]

mappad=map(max,(1,2,3),(4,5,6),(0,5,7,9))
list(mappad)
输出：[4, 5, 7] #多余的参数截断

filter()函数：

含义：函数会根据提供的函数对指定的可迭代对象的每个元素进行运算，将结果为真的进行返回

eg:
mappad=filter(str.islower,"FishC")
list(mappad)
输出：['i', 's', 'h']

**收集参数**
1.参数形式为元组类型
def myfunc(*args):
    print("一共有{}个参数.".format(len(args)))
    print("第2个参数是{}.".format(args[1]))
    print("参数的类型是{}.".format(type(args)))

myfunc(1,2,3,4,5)
结果：一共有5个参数.
	 第2个参数是2.
	 参数的类型是<class 'tuple'>.
        
2.存在关键字参数形式
def myfunc(*args,a,b):
    print(args,a,b)

myfunc(1,2,3,a=5,b=6)
结果：(1, 2, 3) 5 6
3.参数形式为字典类型
def myfunc(**kwargs):
    print(kwargs)
    print(type(kwargs))

myfunc(a=1,b=2,c=3)
结果：{'a': 1, 'b': 2, 'c': 3}
	 <class 'dict'>
4.混合使用
def myfunc(a,*args,**kwargs):
    print(a,args,kwargs)
    
myfunc(1,2,3,b=1,c=1)
结果：1 (2, 3) {'b': 1, 'c': 1}

**解包参数**
args=(1,2,3,4)
def myfunc(a,b,c,d):
    print(a,b,c,d)
    
myfunc(*args)
结果：1 2 3 4
kwargs={"a":1,"b":2,"c":3,"d":4}
myfunc(**kwargs)
结果：1 2 3 4
